## Color Coding

### Day Chips in List
- **Primary Blue**: Selected days ([Mon] [Tue] etc.)
- **Info Blue**: All days indicator ([All days])
- **Consistent**: Same style across all tabs

### Status Chips
- **Green**: Active rules
- **Orange**: Pending/scheduled rules
- **Grey**: Inactive rules

### Priority Indicator
- **Standard text**: Priority number (not color-coded)
